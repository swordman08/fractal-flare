## How can I edit this code?

**Use your preferred IDE**

The only requirement is having Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Follow these steps:

```sh
# Step 1: Make a directory for your project somewhere on PC.
mkdir <PROJECT_FOLDER_NAME>

# Step 2: Navigate to the project directory.
cd <PROJECT_FOLDER_NAME>

# Step 3: Clone the repository using the project's Git URL.
git clone https://github.com/swordman08/fractal-flare.git .

(The . means “clone into the current directory”)

# Step 4: Install the necessary dependencies.
npm i

# Step 5: Start the development server with auto-reloading and an instant preview.
npm run dev

# Step 6: Click on the local host link and use the app in browser.
Local: http://localhost:5173/      or similar



```
